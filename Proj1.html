Lesson 1: Basics of the Web and HTML

The Web

The web is a group of computers that communicate with each other. When a person browses on a computer; the request is send to the server through the Internet. The server finds the page and sends back to the persons computer screen.

HTML (Hypertext Markup Language)

It is the language used to create most of the web pages. It describes the text contents and how the page should look, and how to inclued images, links and all.

Elements and Tags

HTML documents are made of HTML elements. When writing HTML, we use the tags to tell the browser the type of element we are using. Example: element b is to make the text bold. element em is to italicizes the text.

Inline and Block Elements

HTML elements are written in either inline or blocks. Blocks are like boxes that are invisible in the browser.

Box Model

The box model is used to position boxes on a web page and consists of:

Margin: Space between boxes on a page and is invisible.

Border: Outline of a box.

Padding: Area around the center of the box.

Content: The actual contents of the box (like nucleus).

Boxes can be sized in three ways: size by pixels, size by using a percent value, or by using "box-sizing: border-box" for every element

Lesson 2: Creating a Document with HTML

Tree like structure

When writing a document in HTML, we may have different elements inside the main element. It is just like a family tree, parents having children and their children and so on..example;
<parent>
 <child>
  <subchild>.....<subchild>
 <child>
<parent>
In the browser, it shows like boxes inside a box and so on. While writing HTML, we indent the elements in a way so that it is visibly clear where the elements start and end.

Text Editors

When writing a code, programmers use text editors like Codepen and Sublime Text to make coding little easy. It can help them pick a tag from number of options it shows when a programmer types the first letter of the element and so on.

Lesson 3: CSS Style

Avoiding Repetition and Errors

While programming, we may have to use certain style attributes to different HTML elements for the web page. We can do this by writing the same style attribute for different HTML elements again and again. It means repetition and will lead to errors. We can avoid the errors and repetition by assigning the same class to different <div> and then specify the style for that class at one place. Any future changes will be easy too!
For example if we want to change text color on the web page; meaning we have to change the text color attribute at all the places in all the <div>, while as if we have specified a same class for the different <div>, we just need to change the text attribute at one place only.

CSS(Cascading Style Sheets)

Programmers use it to give style to HTML elements. We can use CSS to use different Background colors, use different colors in different blocks, make text bold, change font, italicized and so on.. There are three ways we can do this:

Method 1. At the top of the HTML, we can add head element and write the CSS under the style element.

Method 2. It lets us to stay organized. We can create a separate file with CSS code and link that to the HTML file. We can include the style file name in the head element of the HTML file.

Method3. We write the style inline with HTML. This method should be avoided as it results in lot of repetition and errors.
